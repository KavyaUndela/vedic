# Vedic Learning Platform

## Overview

Vedic Learning Platform is a web application designed to provide users with access to Vedic knowledge and educational resources. The platform includes secure user authentication and content access management using Firebase.

## Features

- User Registration
- Secure Login Authentication
- Firebase Integration
- Personalized User Access
- Educational Content Management
- Responsive Web Interface

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Node.js
- Firebase Authentication
- Firebase Admin SDK

## Project Structure

- login.html – User Login Page
- signup.html – User Registration Page
- home.html – Main Dashboard
- server.js – Backend Server
- firebase-adminsdk.json – Firebase Configuration

## Workflow

1. User creates an account.
2. User logs into the platform.
3. Authentication is verified through Firebase.
4. User accesses Vedic educational resources.
5. Content is managed securely through the backend.
